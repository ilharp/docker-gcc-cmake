# docker-gcc-cmake

gcc+cmake Docker Image, but tagged by os+versions.

```sh
docker pull ilharp/gcc-cmake:ubuntu-18.04
```

## Tag

Tag format: `os-version`

See `build.sh` for available versions.

`gcc` is installed with `build-essential`. CMake is `v3.22.3`.

## Build

Just run `build.sh`.
